# 🧶 Knit Gauge Converter

![Screenshot of app preview](https://github.com/aylinmarie/knit-gauge-converter/blob/c673395bf02047bd0b7dae62d7c5680c5059c8de/app/preview_img)


A lightweight web utility to translate pattern instructions into your personal knitting tension. Perfect for yarn substitutions or matching a specific fit.

## 🛠️ Tech Stack
Framework: [Next.js](https://nextjs.org/) (App Router)
Database & Auth: [Supabase](https://supabase.com/)
AI Engine: [Anthropic Claude API](https://www.anthropic.com/)
